= ECM Sliders

= Installation

Add it to your bundle:

  # Gemfile
  gem 'ecm_sliders2'

Update your bundle:

  > bundle install

Install the module:

  > rails g ecm:sliders:install

Create migrations:

  > rake ecm_sliders_engine:install:migrations

Migrate:

  > rake db:migrate

= Usage

Use the helper to display a slider:

  <%= render_slider 'My Slider' %>
